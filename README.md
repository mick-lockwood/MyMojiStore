# MyMojiStore (2D Web)

A browser-based, 2D simulation game where players manage a trading card shop, open randomized booster packs, and haggle with customers. Built with pure JavaScript and the Phaser.js engine.

## 🎮 Core Gameplay Loop
1. **Acquire Inventory:** Purchase and open booster packs with varying rarity drop rates.
2. **Manage Stock:** Organize cards into binders, display cases, and personal collections.
3. **Trade & Sell:** Interact with NPCs who walk into the shop to buy, sell, or trade cards based on dynamic market values.

## 🛠️ Tech Stack
* **Engine:** [Phaser 3](https://phaser.io/) (Loaded via CDN)
* **Language:** JavaScript (Vanilla / ES6)
* **Graphics:** 2D sprites and UI designed in Adobe Photoshop
* **Hosting:** GitHub Pages
